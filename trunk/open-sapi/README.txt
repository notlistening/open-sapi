To install a development enviroment for the open sapi project please
follow these instructions.

Navigate to the installer folder found in the svn ../trunk/installer/. In this 
folder you will find a script called osapi.run please execute this script in a 
terminal 

$./osapi.run

This script will download and install the required packages to get you
developing or testing on this software project.

Please report bugs to thomaslloyd@yahoo.com or at the project bugtracker:
http://code.google.com/p/open-sapi/issues/

Your feedback is always welcome any help always accepted and don't be affraid to
ask.
